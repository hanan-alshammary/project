#☕ Coffee, Stress, Health, and Sleep Analysis
📌 Project Overview

This project explores the relationship between coffee consumption, stress levels, sleep duration, and sleep quality using Python and Machine Learning techniques.

The project includes:

#Exploratory Data Analysis (EDA)
Data Visualization
Supervised Learning
Unsupervised Learning
📂 Dataset
Dataset: Global Coffee Health Dataset
Source: Kaggle
Records: 10,000
Target Variable: Sleep Quality


#🛠 Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
📊 Exploratory Data Analysis

The project answers questions such as:

Do females or males have better sleep quality?
Which age group drinks the most coffee?
How do sleep hours affect sleep quality?
Which occupation has the highest stress level?
What is the relationship between stress and sleep?


#🤖 Machine Learning Models
Supervised Learning
Logistic Regression
Linear SVC
Unsupervised Learning
K-Means Clustering

#📈 Results
Logistic Regression Accuracy: 99%
Linear SVC Accuracy: 99%
Sleep Hours showed a strong positive correlation with Sleep Quality.
Stress Level showed a strong negative correlation with Sleep Quality.
